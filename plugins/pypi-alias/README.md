# prompt-shield

This is an alias package. Installing `prompt-shield` will install [`prompt-shield-ai`](https://pypi.org/project/prompt-shield-ai/), which contains all the code.

```bash
pip install prompt-shield
# or equivalently:
pip install prompt-shield-ai
```

Both names install the same library. See the [main repository](https://github.com/mthamil107/prompt-shield) for documentation.
